# artint
A python package for easier AI Access. The name means Artificial Intelligence. This project uses text pollinations AI to interpret the user's message.

### everything this project has:
* A class called AI for easier AI Access and creation
* A function "yesorno" to get a yes or no answer from the AI. True is for yes, False is for no
* A function "translate" for translating things into a specific language.
* A custom, easy-to-make Exception that's written in one line
